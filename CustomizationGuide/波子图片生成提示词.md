# 波子 DockCat 图片生成提示词

本文档基于 [DockCat 官方提示词](图片生成提示词.md)，针对波子（布偶猫）的特征做了定制。

## 波子外观特征（生成时必须保留）

上传 `bozi/references/` 中的参考照片后，请在所有提示词中强调以下特征：

- **品种**：布偶猫（Ragdoll），体型圆润、毛发极蓬松
- **毛色**：身体以雪白长毛为主，颈部有浓密围脖
- **重点色**：耳朵、眼周、鼻梁两侧为灰褐色（seal point）色块
- **面部**：额头有白色倒 V 形开脸，粉色小鼻子
- **眼睛**：明亮的天蓝色，圆润有神
- **尾巴**：浓密蓬松的灰褐色大尾巴
- **性格气质**：温顺、好奇、有点呆萌

## 通用美术风格

请使用上传的波子真实照片作为参考图。将这只布偶猫转化为可爱的复古网页宠物游戏风格 2D 数字插画，同时准确保留参考照片中的毛色、花纹、毛长、体型、脸部特征和整体气质。

画风：简化圆润、偏卡通，灵感来自 2000 年代早期网页宠物游戏。干净顺滑轮廓线、略粗深色线稿、简单平涂或柔和混色、少量轻柔阴影。

**重要**：只输出猫咪本身，透明背景。不要地板、阴影、场景、UI、文字。生成时可用纯品红 (#FF00FF) 背景方便后期抠图，再用 `scripts/process_bozi_assets.py` 处理。

## 静态姿势提示词

### 对话站立（优先生成）

```
创建一张透明背景 PNG，内容是参考布偶猫波子面向观众端坐的姿势，适合提醒对话。
保留：雪白长毛、灰褐耳朵和面部色块、蓝色眼睛、白色倒 V 开脸、粉色鼻子、蓬松灰褐尾巴。
表情专注友好。画布上方留透明空间放气泡。不要画气泡。
1024x1024 px，8% 透明边距。
输出：poses/dialogue/stand.png
```

### 侧躺休息

```
波子侧躺休息，眼睛闭合，尾巴自然蜷曲。同样保留布偶猫身份特征。
输出：poses/resting/side.png
```

### 揣手手（面包姿势）

```
波子揣手手休息，爪子收在身下，眼睛闭合，身体圆润紧凑。
输出：poses/resting/loaf.png
```

### 翻肚皮

```
波子仰躺翻肚皮，四爪朝上，表情放松可爱。
输出：poses/resting/spluff.png
```

### 被抱起

```
波子被从腋下抱起，身体因重力略下垂，小爪子露出，表情温和。不要画人类的手。
输出：poses/held/held.png
```

### 伸懒腰

```
波子伸懒腰到最高点：前爪向前伸展，背部拱起，后半身抬高。约 2 秒过渡状态，单张图。
输出：poses/transition/stretch.png
```

## 散步动画

先生成 `bozi_dialogue_v2` 风格的侧面标准图作为身份参考，再生成 4 帧循环：

- `walk_01.png`：第一接触姿势
- `walk_02.png`：经过姿势
- `walk_03.png`：反向接触姿势
- `walk_04.png`：第二经过姿势，自然接回第 1 帧

动作缓慢放松，适合桌面宠物沿程序坞闲逛。只做一个方向，DockCat 会自动镜像。

## 应用图标

- `icon_sleep.png`：波子蜷缩睡在可爱方形软垫上
- `icon_empty.png`：同一个空软垫，没有猫

1024x1024 px，方形图标构图。

## 本地处理流程

1. 用 AI 工具（ChatGPT、Midjourney 等）按上方提示词生成图片，保存到 `.cursor/projects/.../assets/` 或任意目录
2. 在 `scripts/process_bozi_assets.py` 的 `MAPPING` 中登记文件名和目标路径
3. 运行：

```bash
python3 scripts/process_bozi_assets.py
```

脚本会抠除品红背景、统一 1024 画布，并同步到 `CatPacks/bozi/` 和内置 `DefaultCat/`。

4. 在 Xcode 中重新构建，或在设置中点击「加载资源包」
